# Third-party notices

Firearms Vault includes self-hosted open-source libraries so the authenticated
application does not depend on executable code from a public CDN. The deployed
`vendor/manifest.json` records the exact file, version, source URL, license,
size, and SHA-256 checksum for every primary vendored component.

Complete license texts are distributed in `vendor/licenses/`. Components
include SheetJS Community Edition, jsPDF, jsPDF-AutoTable, JSZip, QRCode.js,
Chart.js, Supabase JavaScript Client, Tesseract.js/Tesseract Core, and Inter.
