About

Firearms Vault — Local Edition

Version 1.0

A private, offline manager for your firearms, ammunition, accessories, NFA items and more. All of your data is stored only on this device — it is never uploaded, shared, or sent over the internet.

Because the data lives only here, you are responsible for backups. Use Save to File or Backup Now regularly and keep copies somewhere safe; use Restore from File to load one back.
