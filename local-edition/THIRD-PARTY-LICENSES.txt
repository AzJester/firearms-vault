BUNDLED FOR OFFLINE USE
All of the above except Tesseract.js are bundled (vendored) in `vendor/` as the
unmodified, redistributable library files, so the app runs without internet.
Each library's own license text is included under `vendor/licenses/`.

ONE EXCEPTION — Tesseract.js (optional serial-number OCR) is still loaded from a
CDN, because it additionally downloads a worker, a WebAssembly core, and
language data at runtime. The "Scan Serial" feature therefore needs an internet
connection; every other feature works fully offline. To make OCR offline too,
bundle tesseract.js, tesseract.js-core, and an `eng.traineddata` file and point
the OCR initialization at those local paths.
